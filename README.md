# pyMPI
This is the repository for py-MPI: a simulation toolbox for Magnetic Particle Imaging (ongoing project). This project is based upon code from Shen et al (https://doi.org/10.1016/j.cmpb.2024.108250).

# How to run:
1) Download the code and store it in a folder.
2) Install Python if needed. This project is created on Python 3.9. Minor incompatibility issues may occur in other versions
3) Create a virtual environment in your working directory and run the command:
     pip install -r requirements.txt
  This command will automatically install the required packages
4) Run main
